# 🔥 Ресторан «Огонёк» — Интерактивный Стейк-Хаус

Современный веб-сайт для стейк-хауса «Огонёк», разработанный на чистом стеке PHP + MySQL + Vanilla JS. Проект обеспечивает полный цикл взаимодействия: выбор блюд, корзина, оформление заказа, бронирование столиков, личный кабинет и панель администратора.

## 🌟 Функциональность

### Личный кабинет и авторизация
* Регистрация и авторизация с хэшированием паролей (`bcrypt`).
* SVG-кнопка показа/скрытия пароля.
* Редактирование профиля (email, телефон) через AJAX.
* Адресная книга: добавление, редактирование и удаление нескольких адресов доставки.
* История заказов с кнопкой «Повторить заказ».
* История бронирований с отслеживанием статусов.

### Меню и корзина
* Каталог блюд с фильтрацией по категориям (табы без перезагрузки).
* Поиск по названию внутри активной категории.
* Корзина на `localStorage` + синхронизация с БД для авторизованных пользователей.
* Оформление заказа с выбором сохранённого адреса или вводом нового.

### Бронирование столов
* Кастомный календарь (без сторонних библиотек) с блокировкой прошедших дат.
* Динамическая загрузка доступных временных слотов через API.
* Лимит 10 бронирований на один временной слот.
* Автозаполнение данных для авторизованных пользователей.

### Панель Администратора
* Дашборд: выручка за день/месяц, графики динамики продаж (Chart.js), ТОП-5 блюд.
* Управление меню: добавление блюд с drag-and-drop загрузкой изображений, удаление/скрытие.
* Управление категориями: CRUD с автогенерацией slug из кириллицы.
* Таблица заказов со сменой статуса (Принят → Готовится → Выполнен / Отменен).
* Таблица бронирований со сменой статуса (Ожидает → Подтверждена → Отменена).
* Архивация заказов и бронирований.
* Пагинация для таблицы блюд.

## 🏗️ Архитектура

### Поток данных

```
Браузер (JS)          Сервер (PHP)           База данных (MySQL)
─────────────         ─────────────          ─────────────────────
cart.js       ──►     php/api/cart.php   ──►  cart_items
auth.js       ──►     php/api/auth.php  ──►  users
booking.js    ──►     php/api/booking.php──► bookings
cart.js       ──►     php/api/orders.php ──► orders + order_items
auth.js       ──►     php/api/addresses.php──►user_addresses
admin.js      ──►     php/api/products.php──►products
admin.js      ──►     php/api/categories.php──►categories
admin.js      ──►     php/api/admin.php  ──►orders, bookings (архив)
admin.js      ──►     php/api/upload.php ──► images/ (файловая система)
```

### Принципы
* **Разделение ответственности**: PHP-страницы рендерят начальный HTML, JS-скрипты управляют интерактивностью через Fetch API.
* **REST-подобные API**: каждый эндпоинт принимает `action` через GET/POST, возвращает JSON `{success, message, ...data}`.
* **Двойная валидация**: на клиенте (JS) и на сервере (PHP, класс `Validate`).
* **Компонентный подход**: header, footer и cart-modal вынесены в `php/includes/`.

## 🚀 Установка и запуск

### 1. Размещение файлов
Скопируйте папку `ogonyek` в корень веб-сервера:
```
C:\xampp\htdocs\projects\ogonyek\
```

### 2. Запуск сервера
В **XAMPP Control Panel** запустите модули **Apache** и **MySQL**.

### 3. Импорт базы данных
1. Откройте [phpMyAdmin](http://localhost/phpmyadmin).
2. Создайте базу данных `ogonyok_db` (кодировка `utf8mb4_unicode_ci`).
3. Импортируйте файл `ogonyok_db.sql` из корня проекта.

### 4. Настройка подключения
Если параметры MySQL отличаются от стандартных, отредактируйте `php/config/db.php`:
```php
$host = 'localhost';
$db   = 'ogonyok_db';
$user = 'root';
$pass = '';
```

### 5. Открытие сайта
Перейдите в браузере:
```
http://localhost/projects/ogonyek/
```
